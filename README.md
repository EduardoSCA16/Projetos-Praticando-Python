# 🐍 Exercícios de Python - Alura

Este repositório reúne os exercícios e pequenos projetos desenvolvidos durante meus estudos na formação **Python** da Alura.

O objetivo deste repositório é acompanhar minha evolução na linguagem, praticando lógica de programação, estruturas de controle, funções e, futuramente, Programação Orientada a Objetos (POO).

## 📚 Conteúdo

### 📁 Exercícios
Exercícios de fixação envolvendo:

- Estruturas condicionais (`if`, `elif` e `else`)
- Laços de repetição (`for` e `while`)
- Funções
- Tratamento de exceções
- Manipulação de listas
- Geração de números aleatórios
- Validação de dados

### 📁 Projetos

Pequenos projetos desenvolvidos durante o curso:

- 🎯 Jogo de Adivinhação
- ✂️ Jokenpô
- 🔐 Gerador de Senhas
- 🪪 Validação de CPF
- 🔢 Contador de Vogais
- 📝 Contador de Palavras Longas
- 💰 Calculadora de Gorjeta
- 🧮 Calculadora com Tratamento de Erros
- ✅ Gerenciador de Tarefas
- 💵 Contador de Cédulas Únicas 

## 🚀 Tecnologias

- Python 3

## 🎯 Objetivo

Este repositório faz parte da minha jornada de aprendizado em Python e Backend.

Conforme avanço na formação, novos exercícios e projetos serão adicionados.
